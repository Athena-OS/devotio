# Devotio
Devotio is a secure disk erasure tool based on **DoD  5220.22  M**, **ATA Secure Erase** and **encryption-based destruction**.

It allows to select the specific device to erase or every disk on your system. It involves:
* HDD
* SSD
* USB
* RAM
* Zram
* Swap
